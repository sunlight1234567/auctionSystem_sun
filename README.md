# 在线拍卖系统运行指南

## 1. 环境准备

确保已安装 Python 3.x。建议使用虚拟环境。

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境 (Windows)
.\venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt
```

## 2. 启动系统

在终端中运行：

```bash
python app.py
```

系统启动后，会自动创建 `auction.db` 数据库文件。
默认会创建一个管理员账号：
- 用户名：`admin`
- 密码：`123`

## 3. 本机访问与测试

打开浏览器访问：`http://127.0.0.1:5000`

### 推荐测试流程：
1. **登录管理员** (`admin`/`123`)，进入后台确认无待审核商品，登出。
2. **注册卖家账号** (`seller`/`123`，角色选卖家)。
3. **发布商品**：填写信息，提交。
4. **重新登录管理员**：在“后台管理”中通过审核。
5. **注册买家账号** (`buyer`/`123`，角色选买家)。
6. **竞拍**：进入首页，点击商品，尝试出价。可打开两个不同的浏览器窗口（或隐私窗口），登录不同买家账号互顶价格，观察实时变化。

## 4. 局域网其他设备访问

要让手机或其他电脑访问本系统（充当服务器）：

1. **获取本机IP**：
   打开新的终端，输入 `ipconfig`，找到 "IPv4 地址"，例如 `192.168.1.5`。

2. **确保防火墙允许**：
   如果是 Windows，首次运行 Python可能会弹出防火墙提示，请勾选“允许专用网络”和“公用网络”。
   如果无法访问，尝试暂时关闭防火墙测试。

3. **访问地址**：
   在手机或其他电脑浏览器输入：`http://192.168.1.5:5000` (替换为你实际的IP)。

## 注意事项
- 必须确保所有设备在**同一个 Wi-Fi 或局域网**下。
- 防火墙是局域网连接失败最常见的原因。
